# Analisis de ventas en tiendas de Estados Unidos

Mediante el uso de tablas dinamicas, graficos dinamicos, segmentación de datos y linea de tiempo se pudo analizar los datos de ventas de diversas tiendas pertenecientes a una misma cadena que se encuentran ubicadas a lo largo del territorio estadounidense. Obteniendo como producto final un dashboard con la siguiente estructura:
